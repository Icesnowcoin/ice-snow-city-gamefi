# Ice Snow City Admin Agent - 操作手册

## 快速开始

### 登录系统

1. 访问 https://your-domain.manus.space
2. 点击 "Login with Manus"
3. 使用 Manus OAuth 账户登录
4. 确认您是 Owner 用户

### 首次设置

1. **生成密令**
   - 进入 "密令管理" 页面
   - 点击 "生成新密令"
   - 安全保存返回的密令（仅显示一次）

2. **配置合约参数**
   - 进入 "参数配置" 页面
   - 设置 utilityFeeRate、luxuryGiftRebateRate 等
   - 点击 "保存" 确认

3. **验证系统连接**
   - 进入 "系统状态" 页面
   - 确认所有服务都显示 "Connected"

## 日常操作

### 支付公用事业费

1. 进入 "Agent 操作" > "支付费用"
2. 输入玩家地址和金额
3. 输入密令进行验证
4. 点击 "执行" 提交交易
5. 等待交易确认（通常 10-30 秒）

### 处理奢侈礼物返利

1. 进入 "Agent 操作" > "处理返利"
2. 输入接收者地址和礼物价值
3. 输入密令进行验证
4. 点击 "执行" 提交交易
5. 查看返利金额

### 铸造土地 NFT

1. 进入 "Agent 操作" > "铸造土地"
2. 输入接收者地址
3. 输入坐标 (X, Y)
4. 选择土地类型
5. 输入密令进行验证
6. 点击 "执行" 提交交易

### 铸造房屋 NFT

1. 进入 "Agent 操作" > "铸造房屋"
2. 输入接收者地址
3. 选择土地 Token ID
4. 选择房屋类型
5. 输入装饰哈希（可选）
6. 输入密令进行验证
7. 点击 "执行" 提交交易

## 监控和报告

### 查看系统状态

- **实时监控**: 进入 "监控面板" 查看实时指标
- **事件日志**: 进入 "事件日志" 查看链上事件
- **审计日志**: 进入 "审计日志" 查看所有操作记录

### 生成报告

1. 进入 "报告" 页面
2. 选择报告类型：
   - 操作报告：所有管理员操作
   - 交易报告：所有链上交易
   - 审计报告：审计日志导出
3. 选择时间范围
4. 点击 "生成" 下载报告

### 查看财库信息

1. 进入 "财库" 页面
2. 查看当前 ISC 余额
3. 查看交易历史
4. 导出交易记录

## 密令管理

### 生成新密令

```
操作步骤：
1. 进入 "密令管理"
2. 点击 "生成新密令"
3. 系统生成随机密令
4. 复制并保存密令
5. 点击 "确认" 激活密令
```

**重要**: 密令仅显示一次，请妥善保管！

### 轮换密令

建议每月轮换一次密令以提高安全性：

```
1. 生成新密令（参考上述步骤）
2. 更新所有使用该密令的系统
3. 旧密令将在 7 天后失效
```

### 密令验证

所有需要密令的操作都会进行验证：

```
- 验证方式：Keccak256 哈希比对
- 验证时间：常数时间（防止时序攻击）
- 失败处理：记录失败日志，3 次失败后锁定 15 分钟
```

## 故障排除

### 常见问题

#### Q: 密令验证失败
**A**: 
- 确认密令输入正确（区分大小写）
- 检查密令是否已过期
- 查看审计日志中的失败记录

#### Q: 交易超时
**A**:
- 检查网络连接
- 验证 Gas 费用是否足够
- 查看区块链浏览器确认交易状态

#### Q: 权限不足
**A**:
- 确认您已以 Owner 身份登录
- 检查 Manus OAuth ID 是否正确
- 联系系统管理员

#### Q: 数据不同步
**A**:
- 刷新页面
- 清除浏览器缓存
- 检查网络连接
- 查看服务器日志

## 安全最佳实践

### 密令安全

- ✅ 定期更换密令（建议每月）
- ✅ 使用强密令（系统自动生成）
- ✅ 不要在聊天或邮件中分享密令
- ✅ 不要在公共计算机上保存密令
- ❌ 不要硬编码密令到代码中
- ❌ 不要在浏览器历史中保存密令

### 账户安全

- ✅ 使用强密码保护 Manus 账户
- ✅ 启用两因素认证（如可用）
- ✅ 定期检查登录活动
- ✅ 及时登出系统
- ❌ 不要在公共 WiFi 上访问系统
- ❌ 不要与他人共享账户

### 操作安全

- ✅ 仔细检查操作参数
- ✅ 在执行前确认交易详情
- ✅ 保存所有重要操作的截图
- ✅ 定期审计操作日志
- ❌ 不要在紧急情况下跳过验证
- ❌ 不要批量执行未验证的操作

## 高级功能

### 批量操作

系统支持通过 CSV 文件进行批量操作：

```csv
action,player_address,amount,land_type
payUtilityFee,0x123...,100,
payUtilityFee,0x456...,200,
mintLand,0x789...,0,0,1
```

上传 CSV 文件后系统将：
1. 验证所有行
2. 显示预览
3. 要求密令确认
4. 逐行执行操作

### 自动化任务

可以配置定时任务执行重复操作：

```
例如：每天 00:00 UTC 发放质押奖励
- 任务名称: Daily Staking Rewards
- 频率: 每天
- 时间: 00:00 UTC
- 操作: 调用 distributeStakingRewards()
```

### API 集成

外部系统可以通过 tRPC API 集成：

```typescript
import { trpc } from '@/lib/trpc';

// 支付费用
await trpc.agent.payUtilityFee.mutate({
  player: '0x...',
  amount: '100',
  secretKey: 'your_secret_key'
});
```

详见 [API 文档](./API_DOCUMENTATION.md)

## 数据导出

### 导出格式

系统支持以下导出格式：

- **JSON**: 完整数据结构，用于数据分析
- **CSV**: 电子表格格式，用于 Excel 导入
- **PDF**: 格式化报告，用于打印和存档

### 导出步骤

1. 进入相应页面（事件日志、审计日志等）
2. 点击 "导出" 按钮
3. 选择导出格式
4. 选择时间范围（如适用）
5. 点击 "下载"
