# Ice Snow City GameFi - 两个月开发计划

## 项目目标
在两个月内交付一个完整的试玩版本（MVP），包含核心游戏系统、3-4 个主要场景、50+ NPC 和完整的经济循环。

## 移动优先设计原则（Mobile-First）

**关键要求**：
- 📱 所有 UI 都是移动优先设计
- 👆 所有交互都支持触摸手势
- ⚡ 性能优化针对移动设备
- 📊 流量优化（图片压缩、数据压缩）
- 🎯 屏幕适配（从 320px 到 1920px）
- 🔋 电池优化（减少不必要的计算）

## 技术栈

### 后端
- **框架**：Express.js + tRPC
- **数据库**：MySQL/TiDB
- **缓存**：Redis
- **认证**：JWT + OAuth2

### 前端
- **框架**：React 19
- **样式**：Tailwind CSS 4
- **状态管理**：Zustand
- **UI 组件**：shadcn/ui
- **国际化**：i18n

### 区块链（后续）
- **网络**：BSC（Binance Smart Chain）
- **代币**：ERC-20（ISC）
- **合约**：Solidity
- **工具**：Web3.js / Ethers.js

## 风险管理

### 高风险项
1. **NPC 系统复杂度**
   - 缓解：使用数据驱动的 NPC 系统，可快速生成 NPC
   - 备选：如果时间不足，减少 NPC 数量到 50+

2. **性能优化**
   - 缓解：使用虚拟化和懒加载
   - 备选：如果性能不达标，减少场景数量

3. **美术资源**
   - 缓解：使用占位符和简化的 UI
   - 备选：如果时间充足，后续添加高质量美术

### 中等风险项
1. **区块链集成**
   - 缓解：在 MVP 中使用模拟数据，后续集成真实区块链
   - 备选：优先级降低到 P2

2. **多语言支持**
   - 缓解：使用 i18n 库，快速添加新语言
   - 备选：先支持中英文
