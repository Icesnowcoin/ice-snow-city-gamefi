## 项目概述

### 项目名称

**Ice Snow City**（冰雪城市）- 基于 $icesnowcoin (ISC) 的 2.5D 现代都市社交模拟经营 GameFi

### 项目愿景

创建一个完整的虚拟经济生态系统，将现实世界的经济规律与区块链技术相结合，为玩家提供一个真实、充满机会、充满挑战的虚拟城市生活体验。在这个城市中，ISC 不仅是游戏货币，更是连接个人梦想、社区发展和现实区块链经济的纽带。

### 核心特性

**2.5D 现代都市风格**：游戏采用 3D 半写实卡通风格，融合亚洲审美，呈现一个现代化、充满活力的都市环境。

**完整的经济系统**：ISC 是游戏中的唯一货币和治理代币，支持完整的经济循环，包括生产、交易、投资、消费等。

**丰富的社交系统**：玩家可以与 NPC 和其他玩家进行深度互动，建立关系、合作、竞争。

**真实的生活模拟**：游戏模拟现实生活的各个方面，包括工作、经营、投资、婚姻、社区参与等。

**区块链集成**：游戏与现实世界的区块链网络相连，玩家的游戏资产可以与现实世界的 ISC 代币相互转换。

### 目标玩家

**年龄**：18-45 岁

**兴趣**：经济管理、模拟经营、社交互动、区块链技术

**游戏经验**：中等到高等

## 游戏场景

### 场景总览

**Ice Snow City** 包含 11 个主要游戏场景，每个场景都有独特的视觉风格、NPC 和经济活动。

| 场景名称 | 描述 | 主要 NPC | 主要活动 |
|---------|------|---------|---------|
| ISC 去中心化银行 | 城市的金融中心 | 李行长、王柜员、张投资顾问 | 存款、贷款、投资 |
| ISC 广场 | 城市的商业中心 | 陈广场管理员、刘商人 | 商品交易、社区活动 |
| ISC 社区工会大厅 | 社区治理中心 | 社区工会主席 | 社区治理、任务发布 |
| 咖啡店 | 休闲社交场所 | 李咖啡师、王店长 | 饮品销售、社交 |
| 书店 | 知识传播中心 | 陈书店老板、林图书管理员 | 书籍销售、讲座 |
| 超级市场 | 日用品销售中心 | 张收银员、李超市经理 | 商品销售、采购 |
| 农场 | 农业生产基地 | 王农场主、李农业技术员 | 农业生产、收获销售 |
| 居民楼 | 玩家住宅区 | 各种居民 NPC | 住宅购买、邻里互动 |
| 商场 | 高端购物中心 | 各种店铺老板 | 高端商品销售、时尚 |
| 4S 店 | 汽车销售和维修 | 刘销售顾问、王维修技师 | 车辆销售、维修保养 |
| 公园 | 休闲娱乐场所 | 各种娱乐 NPC | 休闲活动、社交 |

### 场景设计原则

**真实性**：场景基于现实城市的各个功能区，玩家可以找到熟悉的地点和活动。

**多样性**：不同场景提供不同的活动和经济机会，鼓励玩家探索和参与。

**可扩展性**：游戏系统支持添加新场景，随着社区发展而不断扩展。

**视觉一致性**：所有场景都采用相同的 3D 半写实卡通风格，保持视觉一致性。

## ISC 经济系统

### ISC 的获取方式

| 方式 | 描述 | 收益范围 |
|------|------|---------|
| 劳动报酬 | 在各种场景中工作获得报酬 | 10-30 ISC/小时 |
| 任务完成 | 完成 NPC 发布的任务 | 50-2000 ISC |
| 商品销售 | 销售商品或服务 | 变动 |
| 银行利息 | 存入 ISC 获得利息 | 年利率 5-12% |
| 投资收益 | 投资各种产品获得收益 | 年收益率 8-18% |
| 社区活动 | 参加社区活动获得奖励 | 100-500 ISC |

### ISC 的使用方式

| 用途 | 描述 | 成本范围 |
|------|------|---------|
| 生活消费 | 购买食物、衣服等生活用品 | 10-500 ISC |
| 公用事业费 | 支付电费和水费 | 50-200 ISC/月 |
| 房产购买 | 购买房产 | 10,000-1,000,000 ISC |
| 车辆购买 | 购买车辆 | 20,000-500,000 ISC |
| 商业投资 | 投资商业项目 | 变动 |
| 社区贡献 | 捐赠用于社区发展 | 变动 |

### ISC 的价值机制

**供求关系**：ISC 的价值由供求关系决定。

**通货膨胀控制**：通过 ISC 销毁机制、税收机制等控制通货膨胀。

**价格稳定**：通过动态定价、储备基金等保持价格稳定。

### 银行系统

**存款**：玩家可以在银行存入 ISC，获得安全保管和利息收益。

**贷款**：玩家可以申请各种贷款，用于购买资产或投资。

**投资**：玩家可以通过银行进行各种投资。

### 税收系统

**所得税**：年收入的 5-15%，根据收入水平而定。

**房产税**：房价的 1-2% 年度税率。

**交易税**：交易额的 1-2%。

**企业税**：企业利润的 20%。

## 游戏系统

### 角色系统

**角色定制**：玩家可以详细定制自己的角色外观，包括性别、年龄、身体特征、服装等。

**体力系统**：玩家有体力值，需要通过消耗食物和水来补充体力。

**等级系统**：玩家通过完成任务和获得经验升级，解锁新的活动和机会。

### 住宅系统

**房产购买**：玩家可以购买各种房产作为住所或投资。

**房产装修**：玩家可以装修自己的房产，提高舒适度和价值。

**私密空间**：玩家的房产是私密空间，未受邀请的用户无法访问。

### 婚姻系统

**追求伴侣**：玩家可以追求各种可能的伴侣。

**结婚**：当关系达到一定程度时，玩家可以与伴侣结婚。

**家庭财务**：结婚后，玩家和伴侣共同管理家庭的 ISC 资产。

### 商业系统

**开设店铺**：玩家可以开设自己的店铺，经营各种业务。

**员工管理**：玩家可以雇佣员工管理店铺。

**库存管理**：玩家需要管理店铺的库存。

**定价策略**：玩家可以自定商品价格。

### 农业系统

**土地购买**：玩家可以购买土地进行农业生产。

**作物种植**：玩家可以种植各种作物。

**收获销售**：玩家可以收获作物并销售。

**农产品加工**：玩家可以将农产品加工成食品销售。

### 时间系统

**昼夜系统**：游戏内的昼夜交替速度为真实世界的 100 倍，即真实世界的 1 小时 = 游戏内的 100 小时。

**季节系统**：游戏有四个季节，不同季节有不同的活动和机会。

**事件系统**：游戏中会发生各种事件，影响经济和社会。
