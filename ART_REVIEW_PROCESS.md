## 美术审核流程

### 第一阶段：概念设计审核

**提交内容**：
- 场景概念图（2-3 张）
- 美术风格指南
- 配色方案
- 布局草图

**审核要点**：
- ✓ 风格是否符合游戏整体调性
- ✓ 配色是否合理、舒适
- ✓ 布局是否清晰、易于导航
- ✓ 是否符合移动端优化要求

**审核周期**：1-2 天

### 第二阶段：高保真设计审核

**提交内容**：
- 高保真设计稿（UI 模型）
- 交互流程图
- 响应式设计方案
- 动画效果说明

**审核要点**：
- ✓ 设计细节是否精致
- ✓ 交互是否直观
- ✓ 移动端适配是否完善
- ✓ 性能影响是否可控

**审核周期**：2-3 天

### 第三阶段：实现效果审核

**提交内容**：
- 实现的游戏场景
- 性能数据
- 浏览器兼容性测试
- 用户体验测试报告

**审核要点**：
- ✓ 实现效果是否符合设计
- ✓ 性能是否达到要求（FPS > 30）
- ✓ 移动端兼容性是否良好
- ✓ 用户体验是否流畅

**审核周期**：1-2 天

## 场景美术设计方案

### 场景 1：ISC 银行（金融中心）

**美术风格**：现代科技风格
- 主色调：蓝色、银色、绿色
- 配色：#1e3a8a (深蓝) + #e0e7ff (浅蓝) + #10b981 (绿色)
- 氛围：专业、安全、可信

**场景要素**：
- 建筑：现代高楼，玻璃幕墙
- 内部：柜台、ATM、投资屏幕
- NPC：银行职员、客户
- 特效：数字流动、投资图表

**布局**：
```
┌─────────────────────────┐
│  ISC 银行大厅           │
├─────────────────────────┤
│ 柜台 │ 投资区 │ ATM │   │
│      │        │     │   │
│ 咨询区 │ 贵宾室 │ 出口 │
└─────────────────────────┘
```

**移动端优化**：
- 竖屏显示，上下滚动
- 触摸按钮大小 > 44px
- 简化不必要的细节
