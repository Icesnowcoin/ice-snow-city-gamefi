## 1. NPC 日常交易的五个层面

### 1.1 交易层面总览

```
┌─────────────────────────────────────────────────────────────┐
│              NPC 日常交易的五个层面                          │
└─────────────────────────────────────────────────────────────┘

第 1 层: 职业活动 (Professional Activities)
├─ 描述: NPC 根据职业进行的日常工作
├─ 参与者: NPC 与游戏系统
├─ 交易类型: 服务收入、工资支付
├─ 频率: 每天
└─ 示例: 银行家收利息、商人进货

第 2 层: 经济互动 (Economic Interactions)
├─ 描述: NPC 之间的经济往来
├─ 参与者: NPC 与 NPC
├─ 交易类型: 采购、销售、合作
├─ 频率: 每天
└─ 示例: 商人从供应商采购、NPC 之间的交易

第 3 层: 投资活动 (Investment Activities)
├─ 描述: NPC 进行的投资和理财
├─ 参与者: NPC 与投资产品
├─ 交易类型: 存款、投资、收益
├─ 频率: 每周/每月
└─ 示例: 投资顾问推荐投资、NPC 购买理财产品

第 4 层: 消费支出 (Consumption Expenses)
├─ 描述: NPC 的日常生活支出
├─ 参与者: NPC 与商城
├─ 交易类型: 购买商品、支付费用
├─ 频率: 每天
└─ 示例: 购买装备、支付租金、购买食物

第 5 层: 社交活动 (Social Activities)
├─ 描述: NPC 参与的社交和娱乐活动
├─ 参与者: NPC 与其他 NPC/玩家
├─ 交易类型: 赠送、捐赠、参与活动
├─ 频率: 每周
└─ 示例: 参加公会活动、赞助活动、社交消费
```

## 2. 第 1 层: 职业活动 (Professional Activities)

### 2.1 银行家的职业活动

**冷晓雨 (ISC 银行行长)**

```
日常职业活动:

1. 利息收入
   ├─ 来源: 玩家存款的利息
   ├─ 金额: 玩家存款总额 × 0.5% / 天
   ├─ 频率: 每天
   ├─ 示例: 如果玩家存款 100,000 ISC，冷晓雨每天获得 500 ISC
   └─ 月均收入: 15,000 ISC (假设玩家存款 100,000 ISC)

2. 投资管理费
   ├─ 来源: 玩家投资产品的管理费
   ├─ 金额: 玩家投资总额 × 0.2% / 月
   ├─ 频率: 每月
   ├─ 示例: 如果玩家投资 50,000 ISC，冷晓雨每月获得 100 ISC
   └─ 月均收入: 5,000 ISC

3. 贷款利息
   ├─ 来源: 玩家贷款的利息
   ├─ 金额: 玩家贷款总额 × 1% / 月
   ├─ 频率: 每月
   ├─ 示例: 如果玩家贷款 10,000 ISC，冷晓雨每月获得 100 ISC
   └─ 月均收入: 2,000 ISC

月均收入: 15,000 + 5,000 + 2,000 = 22,000 ISC
```

**霜月 (ISC 银行首席柜员)**

```
日常职业活动:

1. 交易手续费
   ├─ 来源: 玩家交易的手续费
   ├─ 金额: 玩家交易总额 × 0.1% / 天
   ├─ 频率: 每天
   ├─ 示例: 如果玩家交易 50,000 ISC，霜月每天获得 50 ISC
   └─ 月均收入: 1,500 ISC

2. 存取款手续费
   ├─ 来源: 玩家存取款的手续费
   ├─ 金额: 每笔交易 0.5 ISC
   ├─ 频率: 每天 (平均 100 笔)
   ├─ 示例: 100 笔 × 0.5 ISC = 50 ISC/天
   └─ 月均收入: 1,500 ISC

3. 工资
   ├─ 来源: 银行支付的工资
   ├─ 金额: 固定 500 ISC / 天
   ├─ 频率: 每天
   └─ 月均收入: 15,000 ISC

月均收入: 1,500 + 1,500 + 15,000 = 18,000 ISC
```

**晨曦 (ISC 投资顾问)**

```
日常职业活动:

1. 投资咨询费
   ├─ 来源: 玩家投资咨询的费用
   ├─ 金额: 每次咨询 100 ISC
   ├─ 频率: 每天 (平均 10 次)
   ├─ 示例: 10 次 × 100 ISC = 1,000 ISC/天
   └─ 月均收入: 30,000 ISC

2. 投资成功奖励
   ├─ 来源: 玩家投资成功的奖励
   ├─ 金额: 玩家投资收益 × 5%
   ├─ 频率: 每周
   ├─ 示例: 如果玩家投资收益 1,000 ISC，晨曦获得 50 ISC
   └─ 月均收入: 5,000 ISC

3. 工资
   ├─ 来源: 银行支付的工资
   ├─ 金额: 固定 800 ISC / 天
   ├─ 频率: 每天
   └─ 月均收入: 24,000 ISC

月均收入: 30,000 + 5,000 + 24,000 = 59,000 ISC
```

### 2.2 商人的职业活动

**云澜 (ISC 商人)**

```
日常职业活动:

1. 商品销售利润

   └─ 月均收入: 30,000 ISC

月均收入: 60,000 + 6,000 + 30,000 = 96,000 ISC
```

## 3. 第 2 层: 经济互动 (Economic Interactions)

### 3.1 NPC 之间的交易

```
NPC 之间的交易示例:

交易 1: 商人向供应商采购
├─ 发起方: 云澜 (商人)
├─ 接收方: 供应商 NPC
├─ 交易金额: 5,000 ISC
├─ 交易类型: 采购商品
├─ 频率: 每周 3 次
└─ 月均交易额: 60,000 ISC

交易 2: 银行家向商人提供贷款
├─ 发起方: 冷晓雨 (银行行长)
├─ 接收方: 云澜 (商人)
├─ 贷款金额: 10,000 ISC
├─ 利息: 1% / 月 = 100 ISC
├─ 频率: 每月 1 次
└─ 月均交易额: 10,000 ISC

交易 3: 投资顾问推荐投资产品
├─ 发起方: 晨曦 (投资顾问)
├─ 接收方: 其他 NPC
├─ 投资金额: 2,000 ISC
├─ 预期收益: 5% / 月 = 100 ISC
├─ 频率: 每周 2 次
└─ 月均交易额: 16,000 ISC

交易 4: NPC 之间的商品交易
├─ 发起方: 各类 NPC
├─ 接收方: 各类 NPC
├─ 交易金额: 1,000 - 5,000 ISC
├─ 交易类型: 商品交易
├─ 频率: 每天
└─ 月均交易额: 100,000 ISC

总月均 NPC 间交易额: 186,000 ISC
```

### 3.2 NPC 交易的经济意义

```
NPC 之间的交易创造了一个完整的经济生态:

1. 供应链形成
   ├─ 供应商 → 商人 → 玩家
   ├─ 每个环节都有利润
   ├─ 形成完整的经济循环
   └─ 增强游戏的经济深度

2. 资金流动
   ├─ 资金在 NPC 之间流动
   ├─ 形成经济活力
   ├─ 增加 ISC 的使用频率
   └─ 提升游戏经济的活跃度

3. 市场形成
   ├─ NPC 之间形成市场
   ├─ 价格会随供求变化
   ├─ 形成真实的经济规律
   └─ 增强游戏的真实感

4. 玩家参与
   ├─ 玩家可以参与 NPC 交易
   ├─ 玩家可以从中获利
   ├─ 增加玩家的参与度
   └─ 提升游戏的可玩性
```

## 4. 第 3 层: 投资活动 (Investment Activities)

### 4.1 NPC 的投资行为

```
NPC 投资活动:

投资类型 1: 理财产品投资
├─ 投资者: 所有 NPC (尤其是高收入 NPC)
├─ 投资金额: 月收入的 20-50%
├─ 预期收益: 2-5% / 月
├─ 示例: 冷晓雨月收入 22,000 ISC，投资 10,000 ISC
└─ 月均投资额: 50,000 ISC

投资类型 2: 商业投资
├─ 投资者: 商人、投资顾问
├─ 投资金额: 月收入的 30-60%
├─ 预期收益: 5-10% / 月
├─ 示例: 云澜月收入 96,000 ISC，投资 50,000 ISC
└─ 月均投资额: 80,000 ISC

投资类型 3: 房产投资
├─ 投资者: 高收入 NPC
├─ 投资金额: 50,000 - 100,000 ISC
├─ 预期收益: 1% / 月 (租金)
├─ 示例: 冷晓雨投资 50,000 ISC 购买房产，每月租金 500 ISC
└─ 月均投资额: 30,000 ISC

投资类型 4: 公会投资
├─ 投资者: 公会会长、高收入 NPC
├─ 投资金额: 10,000 - 50,000 ISC
├─ 预期收益: 3-8% / 月
├─ 示例: 冷峻投资 30,000 ISC 到公会，每月收益 1,500 ISC
└─ 月均投资额: 20,000 ISC

总月均投资额: 180,000 ISC
```

### 4.2 投资的经济意义

```
NPC 投资活动的作用:

1. 资金沉淀
   ├─ 大量 ISC 被投资
   ├─ 减少市场流动性
   ├─ 防止 ISC 过度膨胀
   └─ 维持经济稳定

2. 收益分配
   ├─ 投资产品产生收益
   ├─ 收益分配给投资者
   ├─ 形成新的收入来源
   └─ 增加经济活力

3. 财富积累
   ├─ NPC 通过投资积累财富
   ├─ 形成贫富差异
   ├─ 增加游戏的经济深度
   └─ 提升游戏的真实感

4. 玩家参与
   ├─ 玩家可以参与投资
   ├─ 玩家可以获得投资收益
   ├─ 增加玩家的参与度
   └─ 提升游戏的可玩性
```

```
   ├─ 商城商品销售产生收入
   ├─ 形成经济循环

2. 商品销售
   ├─ NPC 购买商品

3. 经济活力
   ├─ NPC 消费产生经济活动
   ├─ 形成完整的经济循环
   ├─ 增加游戏的经济活力
   └─ 提升游戏的真实感

4. 财富转移
   ├─ NPC 的 ISC 转移到商城
   ├─ 商城的 ISC 转移到玩家
   ├─ 形成财富的流动
   └─ 增加游戏的经济深度
```
