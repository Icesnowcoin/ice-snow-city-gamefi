# Ice Snow City - 游戏场景设计文档

## 1. 场景系统概述

**Ice Snow City** 是一个现代化都市模拟经营游戏，玩家在虚拟城市中通过各种经济活动积累财富。游戏采用 **3D 半写实卡通风格**，所有场景都遵循现代都市美学，融合亚洲审美元素，为玩家提供沉浸式的城市生活体验。

### 核心设计原则

游戏场景设计遵循以下核心原则：

**真实性与虚拟性的平衡**：所有场景都基于真实的城市功能设计，但采用虚拟化、游戏化的视觉表现。场景中的每个元素都有明确的游戏功能和经济意义。

**ISC 经济体系的中心地位**：ISC（$icesnowcoin）是游戏中唯一的治理代币和经济货币。所有场景的设计都围绕 ISC 的流动和使用展开，玩家在每个场景中都能体验到 ISC 的价值和重要性。

**功能性与美观性的统一**：每个场景都具有明确的游戏功能（如交易、生产、社交等），同时保持高品质的视觉设计，让玩家在完成游戏任务的同时享受美观的游戏环境。

**社交性与独立性的结合**：场景既支持多人社交互动，也提供玩家私密空间，满足不同玩家的游戏需求。

## 2. 主要游戏场景设计

### 2.1 ISC 去中心化银行（ISC Decentralized Bank）

**场景定位**：游戏经济的核心枢纽，是 ISC 流动和增值的主要场所。

**场景描述**：这是一座现代化的银行建筑，采用玻璃幕墙和钢铁结构设计，内部装饰豪华而专业。大厅中央有巨大的 ISC 代币符号雕塑，象征 ISC 在游戏经济中的核心地位。银行柜台、VIP 办公室、金库等区域清晰划分，营造出专业而安全的金融氛围。

**主要功能**：

- **ISC 存储与提取**：玩家可以将 ISC 存入银行账户，获得安全保障。
- **利息收益**：玩家存入的 ISC 将根据 APY 合约自动生成利息收益，利息以 ISC 形式发放。
- **贷款服务**：玩家可以用 ISC 或 NFT 资产作为抵押获得贷款，用于扩展业务或购买资产。
- **投资产品**：银行提供多种 ISC 投资产品，包括定期存款、理财产品等，帮助玩家增值财富。
- **财务咨询**：NPC 银行顾问提供财务建议，指导玩家如何有效管理 ISC 资产。

**ISC 经济意义**：ISC 去中心化银行是游戏中 ISC 增值的主要场所。玩家通过在银行存入 ISC 获得利息，这直接体现了 ISC 作为价值存储和增值工具的重要性。银行系统与现实中的 DeFi 协议相结合，让玩家体验到真实的区块链金融体验。

**场景要素**：
- 现代化银行大厅，配有高端装饰
- 多个服务柜台，每个柜台由不同的 NPC 银行员工管理
- VIP 办公室，用于高级客户咨询
- 金库区域，象征资产安全
- 实时显示的 ISC 价格和利率信息屏幕
- 舒适的等候区和休息区

### 2.2 ISC 广场（ISC Plaza）

**场景定位**：游戏的中心社交和交易枢纽，是玩家集聚和信息交流的主要场所。

**场景描述**：这是一个宽敞的现代化城市广场，中央有一座巨大的 ISC 代币纪念碑，象征 ISC 在城市经济中的核心地位。广场周围环绕着各种商业设施、餐饮店铺和休闲区域。广场的设计融合了东方和西方的建筑美学，采用现代化的建筑风格和绿化景观。

**主要功能**：

- **玩家交易市场**：玩家可以在广场上设置摊位，出售自己生产或收集的商品，使用 ISC 进行交易。
- **社交聚集地**：玩家可以在广场上聚集、聊天、组织活动，增强游戏的社交性。
- **信息发布**：游戏系统在广场上发布重要信息、活动公告和市场数据。
- **特殊活动场地**：广场定期举办各种游戏活动，如 ISC 交易竞赛、商业大赛等。
- **拍照打卡地点**：玩家可以在广场拍照留念，分享游戏体验。

**ISC 经济意义**：ISC 广场是 ISC 流动的重要场所。玩家在广场上进行的所有交易都使用 ISC，这让玩家充分体验到 ISC 作为交易媒介的重要性。广场上的交易数据直接反映了 ISC 的市场活跃度和价值。

**场景要素**：
- 宽敞的中央广场，配有 ISC 纪念碑
- 多个商业摊位区域
- 休闲座位和绿化景观
- 信息显示屏，实时显示市场数据和活动信息
- 舞台区域，用于举办特殊活动
- 拍照打卡点和装饰性建筑

### 2.3 ISC 社区工会大厅（ISC Community Guild Hall）

**场景定位**：玩家社区组织和协作的中心，是公会管理和集体经济活动的场所。

**场景描述**：这是一座宏伟的社区中心建筑，内部宽敞而现代化。大厅中央有公会旗帜和成员名单展示，各个功能区域清晰划分。建筑采用开放式设计，鼓励玩家之间的互动和协作。

**主要功能**：

- **公会管理**：公会领导者可以在此管理公会事务、成员、资金等。
- **集体任务**：公会可以发起集体任务，成员完成任务后获得 ISC 奖励。
- **公会金库**：公会可以建立金库，存储公会成员的共同资产（ISC 和 NFT）。
- **协作生产**：公会成员可以协作进行大规模生产活动，如农业、制造业等。
- **公会商业**：公会可以建立自己的商业实体，进行大规模交易活动。

**ISC 经济意义**：ISC 社区工会大厅强调了 ISC 在社区协作中的重要性。通过公会系统，玩家可以集合资源进行大规模经济活动，这体现了 ISC 作为社区经济基础的核心地位。

**场景要素**：
- 宽敞的公会大厅
- 公会成员信息展示区
- 公会金库和资产管理区
- 任务发布和管理区
- 会议室和协作工作区
- 公会商业展示区

### 2.4 咖啡店（Coffee Shop）

**场景定位**：休闲社交场所，提供玩家放松和补充体力的地点。

**场景描述**：这是一家温暖而舒适的现代咖啡店，采用北欧极简风格设计。店内有舒适的座位、咖啡吧台、烘焙区域等。大窗户让自然光线充足，营造出温馨的氛围。店内装饰有艺术作品和绿植，充满生活气息。

**主要功能**：

- **饮品销售**：咖啡店出售各种咖啡、茶饮和饮品，玩家可以用 ISC 购买。
- **食品销售**：提供烘焙食品、甜点等，帮助玩家补充体力。
- **社交空间**：玩家可以在咖啡店聚集、聊天、放松。
- **工作空间**：玩家可以在咖啡店进行一些轻松的工作任务。
- **约会地点**：玩家可以在咖啡店与其他玩家或 NPC 进行约会和社交活动。

**ISC 经济意义**：咖啡店是 ISC 日常消费的场景。玩家每次购买饮品和食品都需要支付 ISC，这让 ISC 成为玩家日常生活的一部分，强化了 ISC 作为游戏经济货币的地位。

- 温暖舒适的座位区

- 烘焙展示区

**主要功能**：

**主要功能**：

**主要功能**：

- **私密空间**：玩家的个人公寓是私密空间，未被邀请的玩家无法进入。

**场景要素**：

## 3. 场景的 ISC 经济集成

### 3.1 ISC 作为通用货币

在所有场景中，ISC 都是唯一的交易货币。玩家在任何场景进行的任何交易、购买或服务支付都使用 ISC。这确保了 ISC 在游戏经济中的中心地位。

### 3.2 ISC 流动路径

ISC 在不同场景之间流动，形成完整的经济循环：

- **生产场景**（农场、工厂）：玩家通过生产活动获得 ISC。
- **交易场景**（ISC 广场、商场）：玩家通过销售商品获得 ISC。
- **消费场景**（咖啡店、超市、4S 店）：玩家消费 ISC 购买商品和服务。
- **投资场景**（ISC 银行）：玩家将 ISC 投资获得收益。
- **社区场景**（工会大厅）：玩家通过社区活动获得和消费 ISC。

### 3.3 场景功能与 ISC 价值的关联

每个场景的设计都强化了 ISC 的价值和重要性：

- **ISC 银行**强调 ISC 的增值功能。
- **ISC 广场**强调 ISC 的交易功能。
- **工会大厅**强调 ISC 的社区功能。
- **生产场景**强调 ISC 的劳动价值。
- **消费场景**强调 ISC 的日常使用。

## 4. 场景设计的视觉风格

所有场景都采用 **3D 半写实卡通风格**（Semi-realistic 3D Cartoon Style），具有以下特征：

**Q 版比例改良**：角色和场景元素采用轻 Q 版比例，头部偏大、身体相对缩小，但保持美观和协调。

**亚洲审美融合**：融合日韩系美型特征，五官精致、眼睛较大，体现现代亚洲审美。

**高品质渲染**：使用现代 3D 引擎的光照、材质和阴影效果，皮肤有柔和光泽，服装质感细腻。

**时尚造型**：所有角色和场景都采用现代都市潮流风格，体现当代审美。

**现代城市美学**：所有场景都基于现代城市设计，采用简洁、高效、美观的建筑和装饰风格。

## 5. 场景的交互性

所有场景都设计为高度交互的环境，玩家可以：

- **与 NPC 互动**：与场景中的 NPC 进行对话、交易、任务等互动。
- **与其他玩家互动**：在公共场景中与其他玩家进行社交、交易、合作等活动。
- **环境互动**：与场景中的物体进行互动，如购物、使用设施等。
- **任务完成**：在场景中完成各种游戏任务和挑战。
- **装饰和自定义**：在私密场景（如公寓）中进行装饰和自定义。

## 6. 场景的昼夜系统

游戏采用 **1:100 的昼夜交替速度**，即真实世界的 1 小时相当于游戏内的 100 小时。这意味着：

- 游戏内的昼夜交替频繁，为玩家提供丰富的视觉变化。
- 不同时间段的场景呈现不同的视觉效果和功能。
- 某些场景在特定时间段才开放（如夜间酒吧）。
- 玩家可以体验完整的日夜循环，增强沉浸感。

### 7.1 电力和水费系统

所有场景都需要消耗电力和水资源。玩家需要定期支付电费和水费（以 ISC 形式），这是持续的 ISC 消费。

### 7.2 场景维护费用

某些场景（如商店、农场）需要定期维护，维护费用以 ISC 形式支付。

### 7.3 场景升级和扩展

玩家可以使用 ISC 升级和扩展场景功能，提升生产效率和收益。
