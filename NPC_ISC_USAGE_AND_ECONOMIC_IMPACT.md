## 3. NPC ISC 对游戏经济体系的影响

```

├─ NPC 消费产生 ISC 流通

├─ 增加经济活力
└─ 玩家可以从中获利

├─ 国库获得补充

├─ NPC 进行投资活动
├─ 形成完整的投资市场
├─ 增加游戏的经济复杂性
└─ 提升游戏的深度

✅ 影响 5: 提升游戏真实感
├─ NPC 有真实的经济活动
├─ 形成真实的经济规律
├─ 增加游戏的沉浸感
└─ 提升玩家体验
```
