# 🎮 冰雪城市 GameFi - 完整启动指南

## 📌 项目概述

**冰雪城市** 是一款结合《模拟人生》玩法和 GameFi 经济系统的区块链游戏。玩家在虚拟冰雪城市中生活、工作、消费、投资，通过 ISC 代币参与游戏经济。

### 核心特色

- 🎭 **模拟人生玩法** - 工作、消费、社交、升级
- 💰 **GameFi 经济** - ISC 代币、质押、挖矿、交易
- 🏠 **房地产系统** - 购买房产、租赁收入
- 🌾 **农业系统** - 种植、收获、销售
- 👥 **NPC 社交** - 200+ NPC、关系系统、任务
- 📊 **实时数据** - 经济指标、排行榜、成就

## 💡 游戏玩法指南

### 1. 工作系统

**如何赚钱：**

1. 选择一个职业（初始选择）
2. 每天工作 8 小时
3. 获得 GC（游戏币）奖励
4. 根据职业和等级，工资不同

**职业对比：**

| 职业 | 基础工资 | 升级奖励 | 特殊技能 |
|------|---------|---------|---------|
| 商人 | 100 GC | +20% | 交易折扣 |
| 农民 | 80 GC | +30% | 农业效率 |
| 工人 | 90 GC | +25% | 建筑速度 |
| 医生 | 120 GC | +15% | 治疗效果 |
| 工程师 | 110 GC | +20% | 维修成本 |

### 2. 消费系统

**日常消费：**

- 🍔 **食物** - 每天 20 GC（维持饥饿度）
- 💧 **饮料** - 每天 10 GC（维持口渴度）
- 🎬 **娱乐** - 每周 30 GC（维持幸福度）
- 🏥 **医疗** - 需要时 50 GC（恢复健康）

**消费提示：**
- 消费 > 收入 → 强制储蓄 → ISC 质押
- 维持平衡 → 游戏体验最佳
- 消费不足 → 属性下降 → 工作效率降低

### 3. 升级系统

**获得经验：**

- 工作 → +10 XP/小时
- 完成任务 → +50 XP
- 社交互动 → +5 XP
- 消费 → +1 XP

**升级奖励：**

- 每升 1 级 → 工资 +5%
- 每升 5 级 → 解锁新职业
- 每升 10 级 → 获得成就徽章
- 每升 20 级 → ISC 空投

### 4. 银行系统

**存款：**
- 存入 GC → 获得利息
- 利息率：5% APY
- 每日结算

**质押 ISC：**
- 质押 ISC → 获得更高利息
- ISC 质押利率：20% APY
- 解锁高级功能

**贷款：**
- 抵押房产 → 获得贷款
- 利率：10% APY
- 用于购买更多资产

### 5. 房地产系统

**购买房产：**

1. 进入房地产菜单
2. 选择房产类型（公寓、别墅、豪宅）
3. 支付首付（房价的 20%）
4. 获得房产所有权

**房产收益：**

- 出租房产 → 每月租金收入
- 房产升值 → 长期增值
- 装修房产 → 增加租金

**房产价格：**

| 类型 | 初始价格 | 月租金 | 升值率 |
|------|---------|--------|--------|
| 公寓 | 5000 GC | 100 GC | 2%/月 |
| 别墅 | 15000 GC | 300 GC | 3%/月 |
| 豪宅 | 50000 GC | 1000 GC | 4%/月 |

### 6. 农业系统

**种植：**

1. 购买农地
2. 选择作物（小麦、玉米、番茄）
3. 投入种子和肥料
4. 等待生长周期

**收获和销售：**

- 生长完成 → 自动收获
- 销售作物 → 获得 GC
- 质量越高 → 价格越高

**作物对比：**

| 作物 | 生长周期 | 产量 | 价格 |
|------|---------|------|------|
| 小麦 | 5 天 | 100 单位 | 1 GC/单位 |
| 玉米 | 7 天 | 80 单位 | 1.5 GC/单位 |
| 番茄 | 10 天 | 60 单位 | 2 GC/单位 |

### 7. NPC 交互

**与 NPC 互动：**

1. 进入 NPC 菜单
2. 选择 NPC
3. 选择互动类型（问候、赠礼、工作、结婚）
4. 获得关系点数

**关系系统：**

- 友好 (0-30) → 基础互动
- 亲密 (31-60) → 特殊任务
- 亲近 (61-90) → 合伙经营
- 深爱 (91-100) → 结婚

**结婚系统：**

- 与 NPC 结婚 → 共享资产
- 生育子女 → 新的游戏角色
- 家庭收入 → 额外收益

### 8. 任务系统

**任务类型：**

- 📋 **日常任务** - 每天重置，简单奖励
- 🎯 **主线任务** - 故事驱动，大量奖励
- 🏆 **成就任务** - 特殊条件，特殊奖励

**完成任务：**

1. 打开任务菜单
2. 选择任务
3. 完成要求
4. 领取奖励

**任务奖励：**

- GC（游戏币）
- XP（经验值）
- 物品
- ISC（稀有任务）

## 💎 ISC 代币经济

### ISC 代币信息

- **总供应量：** 202.6M ISC
- **用途：** 治理、质押、分红、投票
- **获取方式：** 任务、质押、交易

### 质押收益

**质押 ISC 获得：**

1. **利息收入** - 20% APY
2. **分红权益** - 游戏收益分红
3. **投票权** - 参与 DAO 治理
4. **特殊权益** - VIP 功能解锁

### 经济平衡

**消耗 > 产出 设计：**

- 玩家日消耗：120 GC
- 玩家日产出：100 GC
- 差额：-20 GC/天 → 强制储蓄

**储蓄转化：**

- 储蓄 GC → 存入银行 → 获得利息
- 质押 ISC → 获得更高利息 → 参与经济

### Q: 如何快速赚钱？

2. 每天工作 8 小时
3. 完成高奖励任务

### Q: 如何获得 ISC？

**A:**
1. 完成特殊任务
2. 达到特定等级
3. 参与社交活动
4. 交易获得

### Q: 房地产投资值得吗？

**A:** 是的！
- 初始投资：房价的 20%
